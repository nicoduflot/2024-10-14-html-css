Les media

Les media image

En HTML, la balise img permet d'afficher une image. Pour améliorer le référencement et l'accessibilité, des normes et des balise contenantes existent.

Un attribut est nécessaire pour rendre la balise img plus accessible : alt.

l'attribut alt est un texte alternatif de description de l'image : il faut décrire l'image appelée dans la page, on parle ici d'une description de l'image, pas de l'image dans un contexte.

Pour ajouter un contexte amélioré à une image, on peut utiliser le conteneur figure.

Figure permet de lier une description (une legende) avec la balise figcaption à l'image, au niveau sémantique donc de lier cette légeande à l'image au niveau SEO.
